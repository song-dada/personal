- 이미지: 전구1,잔잔한물,출렁이는물,물고기들

1) '전구1'  이미지파일열고
2) Reset All Tools 실행 (모든 툴옵션을 초기화)
     -Move 툴 옵션 : Auto-Select 체크 해제하기.
3) Lasso툴로 필라멘트 쇠부분을 넓게 선택
4) Edit>Fill>Content Aware (= 마우스 오른쪽 클릭 > Fill > Content Aware)
   -선택해제: Edit> Deselect  (= ctrl+d)
5) 남겨진 필라멘트 부분은 도장툴로 (크기:60/하드:0)
   -왼쪽 필라멘트 흔적없는 부분을 Alt+클릭 한후
   -Alt 손 떼고 필라멘트 남겨진 부분을 클릭하여  제거
   -반복하면서 제거
6) 필라멘트 오른쪽 끝 (어두운 부분) 도장툴(크기:20/하드:0)로 제거

7) 새로운 Layer 1추가
    -레이어이름 변경 : 그림자
8) 전구 그림자 만들기위해.. 원형선택툴로  세로로 가늘면서 넓은 타원모양으로 선택하여 검정색으로 채우기
    -선택상태 해제: Select > Deselect (= ctrl+d)
9)Filter>Blur>Gaussian Blue>12 한후
   - 레이어패널:  Opacity 40%
10) 상단메뉴에서, Edit > Free Transform >
      -옵션에서,  X:620px / Y:580px / W:350px / H:70px
      -마무리하려면 엔터키를 누른다.

11) '잔잔한 물' 이미지 불러들여서 
      -Shift키를 누른채로 (배율유지) 전구모양만큼  배율을 축소하고
      -어두운부분이 전구에 위치하도록 이동하고
      -엔터키를 눌러서 이미지 불러오기를 마무리 한후,
      -레이어 블렌드모드: Multifly한후
      -레이어패널 맨밑 아이콘들중에서 > 3번째 아이콘 Layer Mask  클릭
12) 전경색: 검정색설정
    (방법1)
       -브러시: 맨위 첫번째 부드러운 브러시(크기:40픽셀  하드:90%설정)
       -브러시크기  적절하게 변경하면서
       -전구외곽부분을 검정색 드래그하여 가린다.
    (방법2)
       -레이어썸네일창클릭 > 검정색으로 Fill
       -전경색으로 흰색으로 설정하고
       -브러시툴클릭: 크기:200/하드:90
       -전구중앙부분을 흰색으로 문지르면 보이게 된다.
        -브러시 크기 변경해 가면서 문질러서 유리부분 모두 보이도록 한다.
13) 레이어패널아래 맨밑에 7개 작은 아이콘들 중에서 > 4번째 > Adjustment Layer 아이콘클릭> Hue/Saturation>차례로12,43,15 설정후
       -Clip 아이콘클릭하여 아래레이어 이미지에 클리핑마스크 되도록 함.

14) '물고기들'  이미지fmf 별도로 연 후 
      -왼쪽 툴바> Lasso Tool > 물고기 2마리 선택 후
      -디자인 중인 작업창으로 끌고 오면 복사본이 만들어진다.

15) 이미지명 변경: '물고기'
      - 메뉴바: Edit > Free Transform      -
       -옵션바에서   X:560px / H:460px  /  (링크연결하고) X:230px / Y:102.28px  / 각도:13
       -엔터키를 눌러서 완료후
       -레이어  불투명도 80% 

16) '출렁이는 물' 이미지 불러들여
       -옵션바에서 
  X:560px / H:430px  / X:360px / Y:290px  / 각도: 8 
       -마무리하려면 엔터키를 누른다.
17  레이어 블랜드모드 : Overlay

18)  '잔잔한 물' 레이어의 레이어썸네일 창을  Ctrl+클릭하여 선택상태로 만든다.
19) '출렁이는 물' 레이어 층이 선택되어 있는 상태인지 확인하고,
20) 둥그렇게 전구 형태가 select 되어 있을때,
      -레이어패널 맨밑 아이콘들중에서 > 레이어마스크 아이콘 클릭
      -'잔잔한 물' 레이어 외곽에 검정 마스크가 씌워진다.
21) '잔잔한 물'레이어 클릭
22) '출렁이는 물' 레이어에서 '레이어썸네일창'을 클릭하여 선택상태로 만든다.
23) 레이어패널아래 맨밑에 7개 작은 아이콘들 중에서 > 2번째 >Add layer mask하여 선택된 영역만 보이도록 한다.
24) '출렁이는 물' 레이어가 선택되어 있는지 확인하고
25) 레이어패널 하단의 Adjustment Layer 아이콘클릭> Vibrance(색 농도/밝기)>64%,0

26) '잔잔한물' 레이어층을 클릭하고

27) '출렁이는물' 레이어의 '레이어썸네일'창을 Ctrl클릭하여> select 상태로 만든다.

28) '잔잔한물' 레이어의 '마스크썸네일'창을 클릭하고
29) 상단메뉴에서, Select > Invese하여 선택을 반전시킨다.
30) 전경색을 검전색으로 설정한 후,
       -브러시툴 선택
       -출렁이는 물결 윗 부분을 드래그하여 가려지게 한다.
       -상단메뉴에서, select > Deselect 하여 선택상태를 해제한다.

31 레이어층 맨위 > Vibrance 1 레이어 층을 클릭한다.
      -Ctrl+Shift+Alt+E하여 현재까지의 디자인을 새로운 레이어에 복사한다.
32) Ctrl+T 
      -마우스오른쪽 버튼 > Flip Vertical (세로방향 반전)
      -레이어 불투명도 20%
      -옵션바에서  Y:750px
       -마무리하려면 엔터키를 누른다.
33) 지우개툴 선택
      -옵션에서 크기:400 / 하드:0
      -경계 부분을 가로방향으로 <---> 지워준후
      -옵션에서 크기:300 / 하드:100 설정한후
      -윗쪽 전구를  정확하게 클릭 클릭하여  겹쳐진 부분을 지워서 선명하게 만든다.
